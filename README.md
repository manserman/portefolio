## 🚀 Projets en vitrine

### 🔧 [myGrade — Gestion des notes étudiants](https://github.com/manserman/myGrade)

| | |
|---|---|
| **Problème** | Gérer les notes et appréciations de plusieurs étudiants de façon centralisée et sécurisée |
| **Solution** | API REST Spring Boot avec Spring Data MongoDB, gestion des utilisateurs, matières et évaluations |
| **Résultat** | Application backend complète, API documentée, données persistées efficacement |

| | |
|---|---|
